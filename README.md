# Smart Student Schedule Builder

## Description
This project is a Smart Student Schedule Builder developed in Java. It helps university students create conflict-free schedules by selecting courses, lectures, and lab sections step by step using the Builder Design Pattern.

## Features
- Build schedule step by step
- Select courses, lectures, and labs
- Detect time conflicts automatically
- Check section capacity before adding
- Flexible and maintainable design

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Builder Design Pattern

## System Design
The system is based on the Builder Design Pattern, where the schedule is constructed gradually instead of being created in a single step.
